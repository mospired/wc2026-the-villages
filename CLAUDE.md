This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

LED screen inventory project for "The Villages" related to World Cup 2026. File-based storage with Git version control and Git LFS for binary files.

## Repository Structure

- `spreadsheets/` - Excel inventory files (.xlsx)
- `pdfs/` - PDF documents
- `presentations/` - PowerPoint files (.pptx)
- `screenshots/` - Screenshot images (.png, .jpg)

## Git LFS

Binary files (.xlsx, .pdf, .pptx, .png, .jpg) are tracked via Git LFS. See `.gitattributes` for tracked patterns.
